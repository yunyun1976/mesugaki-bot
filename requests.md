# 要件

以下は作りたいソフトウェアの要件です。
技術的に不可能な点や矛盾点、説明不足な点などがある場合は、ユーザーとの対話を通じて本ファイルを編集してください。
そのような点が無くなったら本ファイルの要件を基にソフトウェアを作成してください。

## コンセプト

Discord上で動作するジョークBOTである。

拡張性を考慮し、mainのファイルにはコマンドの動作や汎用的な関数・クラスなどは記述しない。
それらは他のファイルに記述する。

セキュリティを考慮し、一部の強力なコマンドは管理者またはマスターユーザーのみが使用できる。
また動作に必要なトークンはファイルに記述しない。

## 使用技術

- python
- discord.py
- sqlite3

## 作成に際して

### ディレクトリ/ファイル構成

- root/
  - src/
    - libs/
      - (汎用的な関数・クラスを記述したファイル)
    - cogs/
      - (コマンドの動作を記述したファイル)
    - main.py
  - data/
    - barizougon.db
    - abikyoukan.db
    - masters.db
  - READMEファイル

### トークン

本番環境ではトークンを環境変数に登録し、それを参照する。
プログラム作成中は環境変数を参照する代わりに.envファイルを作成する。

### プレフィックス

`/`以降にコマンドを入力する。

## 機能

### メッセージング

- batou
  - barizougon.dbに登録されたフレーズから1つのフレーズを選ぶ
  - フレーズの末尾に"♡"を付け足す
  - コマンドが呼び出されたチャンネルにフレーズを送信する
- wakarase
  - abikyoukan.dbに登録されたフレーズから1つのフレーズを選ぶ
  - フレーズの末尾に"ぉぉぉお♡♡♡"を付け足す
  - コマンドが呼び出されたチャンネルにフレーズを送信する
- add_batou
  - barizougon.dbに新しいフレーズを1つ追加する
  - 既に登録されているフレーズは登録しない
  - 登録できなかった場合、その旨のメッセージを送信する
- add_wakarase
  - abikyoukan.dbに新しいフレーズを1つ追加する
  - 既に登録されているフレーズは登録しない
  - 登録できなかった場合、その旨のメッセージを送信する

### 管理者・マスターユーザー用コマンド

- get_barizougon
  - 管理者またはマスターユーザーのみが使用できる
  - barizougon.dbに登録されたフレーズを全て送信する
  - Discord上でそのままのテキストで表示されるようにする
- get_abikyoukan
  - 管理者またはマスターユーザーのみが使用できる
  - abikyoukan.dbに登録されたフレーズを全て送信する
  - Discord上でそのままのテキストで表示されるようにする
- remove_batou
  - 管理者またはマスターユーザーのみが使用できる
  - 変数でフレーズを指定する
  - 指定したフレーズをbarizougon.db内から削除する
- remove_wakarase
  - 管理者またはマスターユーザーのみが使用できる
  - 変数でフレーズを指定する
  - 指定したフレーズをabikyoukan.db内から削除する

### 管理者用コマンド

- add_master
  - 管理者のみが使用できる
  - masters.dbに新しいマスターユーザーを1人追加する
- check_master
  - 管理者のみが使用できる
  - masters.dbに登録されたマスターユーザーを全て送信する
- remove_master
  - 管理者のみが使用できる
  - masters.dbに登録されたマスターユーザーを1人削除する
